# Фильтры изображений 
## Требования
Для работы проекта вам понадобятся библиотеки Pillow и Numpy, а также Python версии 3.12 и выше.
Для установки требований в терминале можно воспользоваться командой ```pip install -r requirements.txt```
## Как пользоваться
Зайдите в терминал, пропишите строчку ```python main.py <название изображения> <названия фильтров> <аргументы фильтров>```.
В проекте появится новое изображение с припиской changed
## Доступные фильтры
* **black_and_white** - чёрно-белый
* **saturation** *argument* - насыщенность; аргумент, в котором меньше 1.0 изображение становится менее насыщенным, больше 1.0 - более насыщенным
* **brightness** *argument* - яркость; аргумент, в котором меньше 1.0 изображение становится менее ярким, больше 1.0 - более ярким
* **contrast** *argument* - контраст; аргумент, в котором меньше 1.0 мзображение становится менее контрастным, больше 1.0 - более контрастным
* **sharpness**  *argument* - резкость; аргумент, в котором меньше 1.0 изображение становится менее резким, больше 1ю0 - реже
* **contour**  *argument1* *argument2* - контур; чем больше число первого аргумента, тем сильнее контур; второй аргумент определяет цвет контура
* **threshold** *argument* - порог; аргумент явялется числом от 0 до 255, он определяет порог
* **kuwahara** *argument* - кувахара (рисунок); чем больше аргумент больше 1.0, тем больше изображение будет выглядеть как рисунок
* **invert** - инвертирование
* **lut_filter** *argument* - таблица, применяет LUT фильтр (из папки filters)
